# Task Management System

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Visit%20Now-blue)](https://task-management-21.netlify.app/)

A clean, responsive web application for managing daily tasks. It allows users to create, edit, delete, and organize tasks with categories and priority levels. The project showcases modern frontend techniques, intuitive UI/UX, and seamless deployment on Netlify.

## Features
- **Add, edit, delete tasks** with instant UI feedback.
- **Category & priority tagging** for better organization.
- **Responsive design** – works on mobile, tablet, and desktop.
- **Persisted state** using local storage (no backend required).
- **Smooth animations** and modern UI components.
- **Deployed on Netlify** with continuous deployment from the repository.

## Installation & Setup
1. **Clone the repository**
   ```bash
   git clone https://github.com/MukeshKumarCoder/task-management-system.git
   cd task-management-system
   ```
2. **Open the project**
   - No build step is required. Simply open `index.html` in a browser.
   - For a development server (optional):
     ```bash
     npx -y serve .
     ```
3. **Run locally**
   - Navigate to `http://localhost:3000` (or the port shown by the server) to view the app.

## Usage
- **Create a task**: Click the “Add Task” button, fill in the title, description, select a category and priority, then hit **Save**.
- **Edit a task**: Click the pencil icon on a task card, modify fields, and save.
- **Delete a task**: Click the trash icon and confirm deletion.
- **Filter tasks**: Use the category dropdown to view tasks of a specific type.
